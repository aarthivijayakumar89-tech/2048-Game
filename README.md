# 2048 Game (C++)

**Course Project:** MA507P (Oct 2025 – Nov 2025)

This is a console-based implementation of the classic 2048 Game developed in C++ as part of my **MA507P Course Project**.
The project is based on Object-Oriented Programming concepts and simulates the basic gameplay of the classic 2048 puzzle.

## Features
- 4 × 4 game board
- Move tiles using **W, A, S, D**
- Random generation of new tiles (2 or 4)
- Tile merging according to 2048 game rules
- Score calculation
- Win detection when 2048 is reached
- Game over detection when no moves are left
- Quit option with confirmation

## Technologies Used
- C++
- Object-Oriented Programming (OOP)
- Data Structures

## How to Run
1. Compile the program:

```bash
g++ main.cpp -o game
```

2. Run the program:

```bash
./game
```

## Controls
- **W** - Move Up
- **A** - Move Left
- **S** - Move Down
- **D** - Move Right
- **Q** - Quit Game
